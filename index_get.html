What are we demonstrating?

When you open this page, it does not contain any data, but it does contain code that requests a page from a server, on receipt of the page parses it to get some information about it, and then inserts that information into this page below. That is what this page demonstrates.

The request is sent to the PanLinx interface of the PanLex database.

The request asks the interface to look up the English word “demonstration” in the database of about 25 million expressions in about 11,000 language varieties, and to return to us a list of the translations of that expression recorded in the database. This page’s code then counts those translations and inserts the count below.

If the answer appears shortly after you open this page, then the code is working correctly.
